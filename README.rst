LCS API
=======

Longest Common Substring API

Installation
============
.. code-block:: console

   pip install git+https://github.com/cmeza99/lcs-api.git

Usage
=====

CLI
---
.. code-block:: console

   lcs_api --help

Module
------
.. code-block:: console

   python3 -m lcs_api --help
